B. Assignment II

In Question 01 of this assignment, I utilized ITVersity Lab to create and construct a MapReduce job, effectively managing extensive datasets through the MapReduce paradigm. To generate a dataset, I created a data.txt file within the ITVersity Lab's designated folder named 'pRai' (my last name folder), intended for processing via MapReduce.
Moving on to Question 02, Databricks was employed for the task. Appropriate visualizations were generated for both "menu" (VIEW menu.csv) and "sales" (VIEW sales.csv) datasets, focusing on the following key performance indicators (KPIs):
1. Determining the total amount spent by each food category.
2. Calculating the total amount spent by each customer.
3. Analyzing the total amount of sales for each month.
4. Evaluating yearly and quarterly sales trends.
5. Identifying the top-ordered food items.
6. Assessing total sales by each county.
All screenshots of the code are provided in the accompanying PDF document. Furthermore, the complete project code is accessible under the "Data Analysis using PySpark and Visualizations in Databricks" section of the projects, available for viewing on GitHub.

C. Assignment III

In this assignment, for Question 01, I have provided answers to the questions.
